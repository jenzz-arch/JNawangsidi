# Line Follower Robot

An autonomous line follower robot developed for robotics competitions. The robot is designed to navigate a predefined track, complete mission-based challenges, and reach the finish line in the shortest possible time.

## Overview

This project was developed for a line follower robot competition, where the robot was required to autonomously navigate a predefined track while completing a mission within the course.

Using infrared sensors and an Arduino-based control system, the robot continuously detects the track and adjusts its movement in real time. During the competition, the robot was required to push a block into the designated green target area before continuing along the track and crossing the finish line as quickly as possible.

This project demonstrates autonomous navigation, sensor integration, embedded programming, and real-time decision-making in robotics.

---

## Competition Mission

The robot was designed to complete the following tasks autonomously:

* Follow the competition track accurately.
* Navigate curves and intersections.
* Push a block into the designated green target area.
* Resume line following after completing the mission.
* Reach the finish line in the shortest possible time.
